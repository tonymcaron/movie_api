API Documentation page

This web application allows users to explore information about movies, directors and genres. Users have the ability to sign up and create their own favorites list, as well as update their personal profile information. Below you'll find a guide to using this app:

Movie Endpoint Info

Business Logic Description URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
View all movies /movies Returns a list of all movies GET None None JSON array of all movie objects
Movie details by title /movies/:title Returns details about a specific movie GET Movie title None JSON object with movie title, description, director, genre, image, featured
Genre description by name /movies/genre/:genreName Returns description of a movie genre GET Genre name None JSON object with genre name and description
Director information by director name /movies/directors/:directorName Returns information about a specific director GET Director name None JSON object with director bio

User Endpoint Info

Business Logic Description URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Create a new user /users Allows user to create a personal profile POST None Format: JSON object

Example:
{ "username": "John_Doe"; "email": "yourEmail@example.com"; "password": "yourPassword" } 		Format: Confirmation message and JSON object of the new user w/ ID

Example:
{ "id": "12345"; "username": "Jane_Doe"; "email": "yourEmail@example.com"; }
Update username by ID /users/:id Allows user update their username PUT User ID None Format: Confirmation message and JSON object of the new username

Example:
{ "id": "12345"; " new username": "Jane_Doe"; }
Add a movie to favorites list by title /users/:id/:movieTitle Adds a movie to user's favorites list POST ID, movie title None Confirmation message that the specified movie was added to favorites list
Delete a movie from favorites list by title /users/:id/:movieTitle Removes a movie from user's favorites list DELETE ID, movie title None Confirmation message that the specified movie was removed from favorites list
Delete a user profile by ID /users/:id Deletes a user by ID DELETE ID None Confirmation message that the user was successfully deleted
