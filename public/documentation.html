API Documentation page

This web application allows users to explore information about movies, directors and genres. Users have the ability to sign up and create their own favorites list, as well as update their personal profile information. Below you'll find a guide to using this app:

Movie Endpoint Info

Note: URLs are case-sensitive and must use "%20" in place of any space that might be required (such as in a movie title, e.g. "http://localhost:8080/movies/Inside%20Out")

Business Logic URL Description HTTP Method Query Parameters Request Body Data Format Response Body Data Format
View all movies /movies Returns a list of all movies GET None None JSON array of all movie objects
Movie details by title /movies/:title Returns details about a specific movie GET Movie title None JSON object with movie title, description, director, genre, image, featured
Genre description by genre name /movies/genre/:genreName Returns description of a movie genre GET Genre name None JSON object with genre name and description
Director information by director name /movies/directors/:directorName Returns information about a specific director GET Director name None JSON object with director bio

User Endpoint Info

Note: URLs are case-sensitive and must use "%20" in place of any space that might be required (such as in a movie title, e.g. "http://localhost:8080/movies/Inside%20Out")

Business Logic URL Description HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Create a new user /users Allows user to create a personal profile POST None JSON object with user info

Example:
{
"username": "John_Doe", "password": "yourPassword",
"email": "yourEmail@example.com"
"Birthday": "YYYY-MM-DD"
} 		JSON object of the new user w/ ID

Example:
{
"id": "12345",
"username": "John_Doe", "password": "yourPassword",
"email": "yourEmail@example.com"
"Birthday": "YYYY-MM-DD"
}
List of all users /users View a list of all users and info GET None None JSON array of all user objects
View user by username /users/:username View a user's info by username GET Username None JSON object of the specified user

Example:
{
"id": "12345",
"username": "John_Doe", "password": "yourPassword",
"email": "yourEmail@example.com"
}
Update user info by username /users/:username Allows user update their info PUT Username JSON object

Example:
{
"username": "John_Doe", "password": "yourPassword",
"email": "yourEmail@example.com"
} 		JSON object of the updated user info

Example:
{ "id": "12345";
"username": "Jane_Doe";
"Password": "yourNewPassword"
}
Add a movie to favorites list by title /users/:username/
:movieTitle		 Adds a movie to user's favorites list POST ID, movie title None JSON object of user info w/ updated favorites list

Example:
{
"username": "John_Doe", "password": "yourPassword",
"email": "yourEmail@example.com"
"Birthday": "YYYY-MM-DD"
FavoriteMovies: [Movie ID1, MovieID2]
}
Delete a movie from favorites list by title /users/:username/
:movieTitle		 Removes a movie from user's favorites list DELETE ID, movie title None JSON object of user info w/ updated favotires list

Example:
{
"username": "John_Doe", "password": "yourPassword",
"email": "yourEmail@example.com"
"Birthday": "YYYY-MM-DD"
FavoriteMovies: [Movie ID1, MovieID2]
}
Delete a user profile by username /users/:id Deletes a user by ID DELETE ID None Confirmation message that the user was successfully deleted
